## 🍅 kirbs.pomodoro

> *"a pomodoro timer that actually has a soul"*

[![GitHub](https://img.shields.io/badge/view_repo-ff85c8?style=flat-square&logo=github&logoColor=white&labelColor=ff6eab)](https://github.com/avsn17/kirbs.pomodoro)
![Python](https://img.shields.io/badge/Python-85.5%25-ffd6f5?style=flat-square&labelColor=f5c8ff)
![Shell](https://img.shields.io/badge/Shell-14.2%25-c8b4ff?style=flat-square&labelColor=ddd6ff)

A **terminal productivity timer** that's part pomodoro, part space odyssey, part philosophy class. Kirby slides across your screen as you grind. You earn galactic ranks. Kant quotes appear when you need them most.

**✦ highlights**
- 🌌 Animated starfield background that scrolls in real-time
- 🎮 Kirby mascot that slides across the screen as you progress  
- 🏆 Galactic ranking system — from `Space Cadet` → `Galactic Overlord`
- 💬 Wisdom chat — quotes from Brontë, Kant, MJ, Billie mid-session
- 📊 Persistent stats leaderboard saved locally
- 🎵 Music autoplay integration via signal file
- 🐳 Docker + Vercel ready

```bash
# quick start
git clone https://github.com/avsn17/kirbs.pomodoro.git
cd kirbs.pomodoro
chmod +x kirbs.sh && ./kirbs.sh

# or install the alias
echo "alias poyo='cd $(pwd) && python3 pomodoro_timer.py'" >> ~/.zshrc
```

<details>
<summary>⌨️ <b>controls</b></summary>

| key | action |
|:---:|--------|
| `Space` | pause / resume |
| `C` | open wisdom chat |
| `S` | show stats leaderboard |
| `A` | kirby config / settings |
| `M` | toggle music signal |
| `O` | change background color |
| `N` | save & start new session |
| `Q` | save & quit |

</details>

<details>
<summary>🚀 <b>galactic ranking system</b></summary>

| distance | rank |
|:--------:|:----:|

| 500 m | ☄️ Comet Rider |
| 1,000 m | 🚀 Orbit Master |
| 2,500 m | ⭐ Star Pilot |
| 5,000+ m | 🌌 Galactic Overlord |

> 10 meters = 1 minute of focus time

## 🎀 station — study.girl

> *"a terminal-aesthetic study stationery tool with your companion Mochi"*

[![GitHub](https://img.shields.io/badge/view_repo-c8b4ff?style=flat-square&logo=github&logoColor=white&labelColor=b09af5)](https://github.com/avsn17/station)
[![Live Demo](https://img.shields.io/badge/live_demo-✦-ffd6a5?style=flat-square&labelColor=ffbe7a)](https://station-smoky-ten.vercel.app)
![HTML](https://img.shields.io/badge/HTML-93.5%25-ffd6f5?style=flat-square&labelColor=ffc8f0)
![CSS](https://img.shields.io/badge/CSS-3.8%25-c8b4ff?style=flat-square&labelColor=ddd6ff)

A **browser-based study companion** with a pixel-art mascot named Mochi who reacts to everything you do. No build step, no dependencies — just open `index.html` and vibe. Built for students who study better when their tools feel alive.

**✦ highlights**
- 🐱 Interactive SVG mascot Mochi who reacts to your mood, tasks, and timer
- ✦ Boot sequence — animated startup with system checks
- 📝 Notes — write, tag & save (export to `.txt`)
- ⏱️ Pomodoro / focus timer with circular progress ring
- 🃏 Flashcards — 3D flip animation, shuffle, navigate
- ✅ Tasks — study checklist with satisfying checkoffs
- 💜 Mood tracker — Mochi responds empathetically
- ✧ Floating ambient sparkle background
- 📱 Deploy to GitHub Pages in 3 clicks

```bash
# local use — seriously just this
open index.html

# or deploy to GitHub Pages
# Settings → Pages → Deploy from branch → main → / (root)
```
